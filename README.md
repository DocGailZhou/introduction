## About Dr. Gail Zhou

Dr. Gail Zhou is a Microsoft Principal Software Engineer who has led 60+ customer engagements, published 12 GitHub solution accelerators, and is an inventor with two awarded US patents.  

Dr. Gail Zhou holds a PhD in Electrical and Computer Engineering from the University of Tennessee. Her PhD dissertation pioneered AI/ML applications in process control, using MATLAB to simulate systems and generate thousands of training datasets for ML algorithms.
